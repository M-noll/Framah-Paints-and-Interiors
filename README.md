# Framah Paints and Interior Website
A modern, responsive business website developed for Framah Paints and Interior, a service provider in Meru, Kenya, specializing in Painting Services, interior design,hardware supply and home maintenance.

### **🚀 Features**
* Fully Responsive Navigation: Custom-built mobile toggle menu for seamless browsing on all devices.

* Dynamic Gallery: An interactive portfolio section that pulls project data from a JavaScript object, featuring a "See More" toggle and a custom-built Lightbox for viewing high-resolution project images.

* Service Showcase: A categorized list of professional services including Gypsum design, painting, tile works, and more.

* Booking Integration: A functional consultation form powered by Formspree.

* WhatsApp Integration: A floating action button for direct customer communication.
    
### **⚙️ Tech Stack**
* **Frontend**: HTML5, CSS3 (Custom Grid and Flexbox layouts).

* **Interactivity**: Vanilla JavaScript (ES6+) for DOM manipulation and gallery logic.

* **Tools**: Git/GitHub for version control.
